# 小红书标题生成器

这是一个最简单、可直接运行的静态网页小项目：
输入一个主题（比如“自由职业”），点击按钮后，自动生成 10 个小红书风格标题。

## 这个项目要做什么

帮助你快速生成有情绪、有反差、有吸引力（但不油腻）的标题灵感，用于练习内容创作。

## 需要哪些文件

- `index.html`：网页结构（输入框、按钮、结果区）
- `style.css`：网页样式（配色、排版、按钮外观）
- `script.js`：网页交互逻辑（点击按钮后生成标题）

## Windows 小白查看成果（最简单方式）

1. 下载这个项目代码（ZIP）
2. 解压 ZIP 文件
3. 打开解压后的文件夹
4. 直接双击 `index.html`
5. 浏览器打开后，在输入框输入主题，点击“生成标题”即可看到结果
